# cgodo
##cgodo是什么?
集合了大量常用的工具类。

##cgodo有哪些功能？

* 工具类
    *  时间工具类
    *  Bean工具
    *  数据库工具
    *  等
* springmvc字段解析器
* 文件上传组建
* `安全`组建
